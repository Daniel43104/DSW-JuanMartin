Git


Es un sistema de control de versiones gratuito, el cual permite trabajar en proyectos grandes, pequeños de forma individual o colaborativa desde cualquier lugar, es un repositorio que permite alojar el codigo fuente online y derivar ramas para su trabajo sin modificar el proyecto principal asi mismo permite regresar a una version anterior.
