Bienvenida

Este sitio está orientado y diseñado con fines educativos. esperamos sea útil y de tu agrado se busca exponer y aclarar de la mejor manera lo que es un sistema de control de versiones distribuido y de código abierto como lo es GIT. Así mismo conocer la plataforma de desarrollo colaborativo GITHUB también conocida como la a red social de los desarrolladores.

Introduccion ¿Que es y Para que sirve?

Un sistema de control de versiones y una plataforma de interaccion y desarrollo es una opcion ideal para trabajar en equipo de forma profecional.
